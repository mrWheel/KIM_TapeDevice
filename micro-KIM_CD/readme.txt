This disk contains programs for the KIM-1 and the Micro-KIM. It also contains books and user guides for the KIM-1. Some of the books may be useful for the Micro-KIM so they are included on this CD. 

The USB drivers are for the USB to serial converter cable offered for the Micro-KIM at Briel Computers. This is an optional cable.

KIMPAPER is a program is a program by Hans Otten that converts .bin hex files into papertape format text files for the KIM-1 and the Micro-KIM.
